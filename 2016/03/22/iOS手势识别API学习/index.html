iOS手势识别API学习

发表于 2016-03-22 06:32   |   分类于 API 学习   |  

前言

在iOS中,苹果给我们提供了很多相当丰富的手势操作,而我们也只需要进行简单的调用,就可以直接使用.并且还可以监控手势状态

UIGestureRecognizer 手势识别器

UIGestureRecognizer 是一个抽象类,iOS中所有的手势都是它的子类 ,有以下子类 :
UIPanGestureRecognizer.h –> 拖动,平移
UIScreenEdgePanGestureRecognizer –> 屏幕边缘平移
UITapGestureRecognizer.h –> 轻拍
UISwipeGestureRecognizer.h –> 轻扫
UILongPressGestureRecognizer.h –> 长按
UIPinchGestureRecognizer.h –> 捏合
UIRotationGestureRecognizer.h –> 旋转
在iOS中监控手势的代理 : UIGestureRecognizer Delegate
