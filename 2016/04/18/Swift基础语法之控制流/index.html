Swift基础语法之控制流

发表于 2016-04-18 09:43   |   分类于 Swift语法   |  

控制流

使用 if 和 switch 来进行条件操作，使用 for-in 、 for 、 while 和 repeat-while 来进行循环。包裹条件和循环变量括号可以省略，但是语句体的大括号是必须的

if 和 for-in 语句

遍历一个数组中的所有数字,大于50自增3,否则自增1

Swift和OC在条件语句上的判断有些区别,OC中允许使用某个变量放到条件语句的判断式中进行判断,系统会自动判断是否为 nil(0) 或 有值.但是Swift中if语句的条件判断式必须是一个表达式,而不会隐形的和nil或0去做判断

你可以一起使用 if 和 let 来处理值缺失的情况。这些值可由可选值来代表。一个可选的值是一个具体的值或者是 nil 以表示值缺失。在类型后面加一个问号来标记这个变量的值是可选的

如果变量的可选值是 nil，条件会判断为 false ，大括号中的代码会被跳过。如果不是 nil ，会将值赋给 let 后面的常量，这样代码块中就可以使用这个值了。
另一种处理可选值的方法是通过使用 ?? 操作符来提供一个默认值。如果可选值缺失的话，可以使用默认值来代替

你可以使用for-in来遍历字典，需要两个变量来表示每个键值对。字典是一个无序的集合，所以他们的键和值以任意顺序迭代结束

使用while来重复运行一段代码直到不满足条件。循环条件也可以在结尾，保证能至少循环一次
